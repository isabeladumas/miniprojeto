> Projeto focado na análise de dados de saúde e desenvolvimento de algoritmos de Machine Learning para predição e triagem de risco de câncer de pulmão.

## ⚙️ Funcionalidades (Features)

*   **Limpeza e Formatação de Dados:** Tratamento completo da base de dados para garantir consistência e qualidade na modelagem.
*   **Análise Exploratória de Dados (EDA):** Estudo aprofundado do comportamento das variáveis do DataFrame e suas correlações.
*   **Treinamento de Modelos Preditivos:** Implementação e avaliação de desempenho utilizando os algoritmos de **Random Forest** (Floresta Aleatória) e **Regressão Logística**.
*   **Interface no Streamlit (`app.py`):** Estruturação e exposição das funcionalidades em um painel web intuitivo.
*   **Simulador de Previsão:** Questionário interativo no Streamlit que calcula e prevê o risco de câncer de pulmão baseado nas respostas fornecidas pelo usuário em tempo real.
